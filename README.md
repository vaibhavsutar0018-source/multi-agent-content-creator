# Multi-Agent Content Creation Crew

## 📌 Description
This project demonstrates a multi-agent system where different agents collaborate to generate content.

## 🤖 Agents Used
- Research Agent → Generates key points
- Writer Agent → Writes blog content
- Editor Agent → Improves and formats content

## ⚙️ How it Works
User enters a topic → agents process it → final content is generated.

## 🛠️ Technologies Used
- Python
## 🚀 Output
Generates structured blog content based on user input.

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Click Run
3. Enter a topic
4. Get output
